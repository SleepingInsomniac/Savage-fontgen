Svg icon font generator
====

##Requirements

* Ruby
* Python
* pyyaml
* fontforge

##Install

###OSX:

* `bundle install`
* `brew install fontforge`
* `pip install pyyaml`


##Run

* Edit conf/config.yaml
* place svgs in the glyphs folder
* `./savage.rb` (may need to make executable)
* Review output in 'out' folder
